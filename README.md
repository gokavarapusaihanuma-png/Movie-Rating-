# Movie Rating App

A React application built with Vite that displays trending movies using the TMDB API.
Users can add movies to a watchlist, search, filter by genre, and sort by rating.

## Features
- Trending movies from TMDB API
- Add / remove movies from watchlist
- Search by movie name or genre
- Sort movies by rating
- Data persistence using localStorage

## How to run this project locally
## How to run this project locally

1. Clone the repository  
   git clone <repository-url>

2. Go into the project folder  
   cd Movie-Rating-

3. Install dependencies  
   npm install

4. Start the development server  
   npm run dev

5. Open the browser and visit  
   http://localhost:5173


## Tech Stack
- React
- Vite
- Tailwind CSS
- Axios
- TMDB API
